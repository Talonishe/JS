# Підсумкова модульна робота: Основи програмування JavaScript
---

### Завдання 1. `summarizeNumbers(numbers)`
- Використання циклу `for...of`
- Обробка порожнього масиву
- Розрахунок статистики

**Файл:** [`js-1/`](js-1//)

### Завдання 2. `processProducts(products)`
- Використання `filter`, `map`, `reduce`
- Робота з об'єктами
- Обробка крайніх випадків

**Файл:** [`js-2/`](js-2//)

### Завдання 3. `createApiClient(baseUrl)`
- Замикання (closure)
- Async/await + fetch
- Приватний лічильник запитів
- Обробка помилок

**Файл:** [`js-3/`](js-3//)

### Завдання 4. TodoList (ООП + DOM)
- Класи `Task` та `TodoList`
- Інкапсуляція
- Делегування подій
- Інтерактивний інтерфейс

**Папка:** [`js-4/`](js-4//)

---

## Як запустити

1. Завантажте файли.
2. Для завдання 1 відкрийте `js-1/index.html` у браузері.
3. Для завдання 2 відкрийте `js-2/index.html` у браузері.
4. Для завдання 3 відкрийте `js-3/index.html` у браузері.
5. Для завдання 4 відкрийте `js-4/index.html` у браузері.
